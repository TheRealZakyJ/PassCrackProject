### Command Line Arguments

        -p = Plain Text
        
        -h = hashed

        -m = MD5 Hashed
     
        -b = BCrppt
        
        -s = sha256 hashed


        brute = brute force attack

        dict = dictionary attack
        
  



### Formating

    - java PassCracking thomas dict
